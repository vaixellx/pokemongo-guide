วิธีต่อสู้ GO Rocket Grunts

GO Rocket Grunts คือเหล่าวายร้ายที่จะโผล่มาตาม PokéStop (ทำให้กลายเป็นสีดำ) หรือบางครั้งก็จะนั่งบอลลูนสีดำมา โดยวายร้ายเหล่านี้จะมีชุดของโปเกมอนที่เลือกใช้ทั้งทีมเป็น "ประเภทเดียวกัน"

ดังนั้นการรู้ว่าวายร้ายเหล่านี้ใช้โปเกมอนชุดไหนและเลือกโปเกมอนที่ "ชนะทาง" เข้าไปสู้จะทำให้ชนะได้อย่างง่ายดาย ส่วนวิธีการดูว่าทีมวายร้ายใช้โปเกมอนชุดไหนนั้นสามารถสังเกตได้จากประโยคที่ พวกเขาพูดในหน้าที่มีปุ่ม Battle (ดูตัวอย่าง)

โดยบทความนี้ได้รวบรวมประโยคที่เหล่าวายร้ายใช้คู่กับชุดโปเกมอน (ตัวใดตัวหนึ่งจากในลิสต์) และประเภทที่ควรเลือกเข้าไปสู้ไว้ให้แล้ว
