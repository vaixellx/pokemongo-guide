ประเภทของโปเกมอน

โปเกมอนแต่ละชนิดจะมีประเภท (บางชนิดอาจะมีได้ถึง 2 ประเภท) การเลือกประเภทของโปเกมอน และประเภทท่าโจมตีที่ชนะทางจะส่งผลให้สามารถชนะการต่อสู้ได้ง่ายยิ่งขึ้น
